Trigger Distance
The threshold, from each site, at wich the controls will be hidden.
All values in "px". Use "-1" to disable one site.
Use Hotkey
Hotkey
The hotkey that hides the controls.
Invert Trigger
Inverts the function of the hotkey and the mouse trigger so that they make the controls visible instead of hiding them. This also hides the full screen controls by default.
Try 70 as trigger distance for the top and bottom.
Only Fullscreen
Only hide the controls when the YouTube video is in fullscreen mode.
